ノベルゲームとScriptをつなげることを試す。
ノベルゲームのフォームの修正。

スクリプトの書き方はほとんど吉里吉里のタグリファレンスを流用。

スクリプトの開き方は、ファイルをダブルクリックして、開くプログラムの設定をメモ帳にしたら、普通に読み込める。
//ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー//
//																    //
//					    スクリプトの説明							//
//											   					    //
//ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー//
タグ(コマンド)は一行につき一個とする。


＝＝＝画像表示＝＝＝
[image storage="イメージファイル" layer=タイプ (top=座標y left=座標x)]
〜〜〜入力必須〜〜〜
このタイプは以下の通りである。
base:背景レイヤー
   0 : 前景レイヤー0
   1 : 前景レイヤー1
   2 : 前景レイヤー2
   3 : メッセージレイヤー
以上をlayerで指定した画像イメージをstorageで指定したイメージファイルに差し替える

〜〜〜入力任意〜〜〜
topは画像左上のy座標を指定します
leftは画像左上のx座標を指定します


＝＝＝任意の時間一時停止＝＝＝
[wait time=時間(単位不明)]
時間を待つ
〜〜〜入力必須〜〜〜
timeで指定した時間ストップする。


＝＝＝セリフ関係＝＝＝
[r]
改行する

[l]
クリック待ち

[cm]
セリフの文字をクリアにする


＝＝＝演出関係＝＝＝
[trans storage="遷移させたいイメージファイル名" method=メソッド名 time=何秒間 (from=どの方向から)]

〜〜〜入力必須〜〜〜
storage : 遷移先のイメージファイル名を指定。
method  : メソッド名は以下に示すものに限る。
    crossfade : クロスフォードで画像を切り替える
    scroll    : スクロールで画像を切り替える
    fadein    : フェードインで画像を切り替える
				フェードインは未実装...
time   : 遷移するのにかかる時間を指定する。

〜〜〜入力任意〜〜〜
from:どの方向から遷移させるか最初の位置を以下から選択。
    top    : 上から下に切り替える。
    bottom : 下から上に切り替える。
    right  : 右から左に切り替える。
    left   : 左から右に切り替える。
なお初期値はbottomになっている。


＝＝＝演出関係＝＝＝
[link target="遷移先にリンク先" value="選択肢に表示される文字列"]

〜〜〜入力必須〜〜〜
target:遷移先のリンク(リンクの定義はその他で)を(アスタリスクをつけて)指定
value:選択に描写される文字を設定
※このタグは必ず次の[linkstop]を最後に示す必要がある。
※逆にこの場所以外でやるとうまく動かない。

[linkstop]
ここまでが選択しであり、選択肢ありのラベル(リンク)の目印になる。
選択肢なしのラベルで使用するとうまく動かない可能性がる。


＝＝＝その他＝＝＝
;
冒頭部分にセミコロンがある行は読み飛ばされる。
つまり、コメントを描くときにセミコロンを一番最初に示す。

*
冒頭にアスタリスクがある単語はそこは遷移先のリンク名となる
主に、これは選択肢関連のtarget要素に使うリンク先で扱う
